# coding-puzzlers

A place to view, solve, and run common coding puzzles.  

## Getting Started

### Clone
`git clone https://github.com/eallen10/coding-puzzlers.git`

### Install
`npm i`

### Run All Problems
`npm run test`

### Run Specific Problem
`npm run test -- -i '<path_to_problem_spec>`
